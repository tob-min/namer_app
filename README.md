# Namer App

A simple Flutter app built while learning the basics of Flutter and Dart.

## About

Generates random word pairs and lets you save your favourites. Built by following the [Flutter codelab](https://docs.flutter.dev/get-started/codelab) as an introduction to Flutter development.

## Features

- Generates random word pairs
- Like/unlike word pairs to save them as favourites
- View your saved favourites in a separate page
- Responsive layout with a navigation rail

## Tech

- [Flutter](https://flutter.dev/)
- [provider](https://pub.dev/packages/provider) — state management
- [english_words](https://pub.dev/packages/english_words) — random word pair generation
